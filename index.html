Projects

AI Mouse Using Hand Gesture

This Virtual Mouse allows users to control their computer cursor using hand gestures detected through a webcam. Leveraging OpenCV for real-time image processing, Mediapipe for hand tracking, and PyAutoGUI for simulating mouse actions, the system provides a touchless, hygienic, and accessible alternative to traditional input devices.

Voice Controlled Home System

This project develops a Voice Controlled Smart Home System using the ESP-32 microcontroller and 4 relay modules to automate home appliances. User voice commands are converted to text via Speech-to-Text (STT) technology, matched against AIML patterns to determine actions, and executed by triggering relays to turn devices on or off.

IELTS Band Assessment Tool

We are developing an NLP-based tool for individuals preparing for the IELTS test. This tool includes reading, listening, writing, and speaking tests. Users can take a practice test, which we then evaluate to provide an estimated IELTS band score. This helps users understand their current proficiency level and identify areas for improvement.

Pakistan-24 Election Prediction

As the political landscape in Pakistan moves toward the 2024 general elections, different phases have been highlighted using a systematic, data-driven approach. COVID-19 Diagnose

A Convolutional Neural Network (CNN) for diagnosing COVID-19 using a dataset of chest X-ray images. The CNN model is built using the Keras library with a TensorFlow backend and this is further advanced by using Transfer learning. To make it work ideal we implement Grid Search Cross Validation.

AI Chatbot with Neo4j

The Chatbot with Neo4j Graph Database project demonstrates the integration of a conversational AI chatbot with a Neo4j graph database. By combining AIML, NLP techniques, and a powerful graph database, the project provides a personalized and context-aware conversational experience. The project shows graph databases for storing and querying relationships between users, enabling the chatbot to provide more meaningful responses.
